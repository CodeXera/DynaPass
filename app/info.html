Hello welcome to DynaPass by XeraCode.

I want to tell you what this project is about. This starts with password generators that do not know if they save or store the generated texts in the cloud. Or they are sent to a local database on the page where all the texts are stored.

If they know about these types of things they will understand it quickly, no matter how secure the password or the page is, if one does not know what the page does with that generated text, it can be dangerous for the user. and therefore lose your Gmail account or some important app. And that is why I am creating and publishing this project on GitHub that generates random texts to use as a password.
